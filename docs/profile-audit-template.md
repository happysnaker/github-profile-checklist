# GitHub Profile Audit Template

Use this template when you want to review a GitHub profile as if you were:

- a hiring manager
- an interviewer
- a senior engineer doing a quick credibility scan
- a founder / recruiter deciding whether to click deeper

The goal is not to be “nice”. The goal is to answer:

1. **What story does this profile tell in 10 seconds?**
2. **What weak signals are currently dragging the profile down?**
3. **What is the smallest set of changes that would improve first impression quickly?**
